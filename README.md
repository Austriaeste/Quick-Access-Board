# 共有フォルダランチャー（HTA）

Windows 環境向けの **軽量 HTA ランチャー**です。  
社内ファイルサーバ上の共有フォルダを、ワンクリックで開くことを目的としています。

- 管理者権限不要
- インストール不要（単一ファイル）
- Windows ログオン資格情報をそのまま利用
- タスクバーにピン留め可能

---

## 特徴

- **3行 × 5列（計15個）**の共有フォルダランチャー
- Explorer を直接起動するため動作が高速
- 画面右下に **余白なしで固定表示**
- HTA（mshta.exe）ベースのため配布が容易

---

## 動作環境

- Windows 10 / Windows 11
- Internet Explorer コンポーネント有効環境（HTA動作要件）
- 社内ネットワーク（UNCパス接続可能）

---

## ファイル構成

```text
Launcher.hta   ← このファイルのみ
````

---

## セットアップ手順

### 1. ファイルを保存

以下の内容を `Launcher.hta` という名前で保存します。

---

### 2. 配置場所（推奨）

以下のような **固定パス**に配置することを推奨します。

```text
C:\Tools\Launcher\Launcher.hta
```

---

## 使い方

* ボタンをクリックすると、対応する共有フォルダを
  **Windows Explorer で直接開きます**
* 共有先は以下の形式（UNCパス）で指定されています

```text
\\192.168.1.10\share01
\\192.168.1.10\share02
...
```

必要に応じて、IPアドレスや共有名を自環境に合わせて変更してください。

---

## タスクバーにピン留めする方法（重要）

HTA ファイルは **そのままではタスクバーにピン留めできません**。
以下の手順でショートカット経由で設定します。

### 1. ショートカット作成

`Launcher.hta` を右クリック → **ショートカットの作成**

---

### 2. ショートカットの設定

ショートカットを右クリック → **プロパティ**

#### リンク先

```text
"C:\Windows\System32\mshta.exe" "C:\Tools\Launcher\Launcher.hta"
```

※ 両方 **必ずダブルクォートを付けてください**

#### 作業フォルダ（任意だが推奨）

```text
C:\Tools\Launcher
```

---

### 3. タスクバーにピン留め

設定した **ショートカット**を右クリック
→ **タスクバーにピン留め**
